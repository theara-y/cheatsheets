Constructors

  • Name of the constructor must be the same as its class name.
  • Constructors have no explict return type.
  • Constructors cannot be abstract, static, final, synchronized.

Types of constructors

  • Default
  • Parameterized

Default Constructors

  • No arguments
  • Implicity created
  • Calls super()
  • Disappears if there are constructors already defined

Parameterized constructors

  • Has arguments/parameters
  • Used to customize the object

Constructor Overloading

When there are multiple constructors in a class

Multiple Inheritance

Multiple inheritance is NOT supported in Java.

Multiple inheritance is a diamond type inheritance pattern. It leads to ambiguity which is why it's not supported.

Overloading

Duplicate methods within a class.

Overriding

Duplicate methods in parent and child classes.

The names and parameters should match.

Can a static method be overridden?

No.
